## 推荐最小起步

四个文件，最小起步就够了：

1. `AGENTS.md` 或 `CLAUDE.md` — 根指令文件
2. `feature_list.json` — 功能状态文件
3. `claude-progress.md` — 进度记录
4. `init.sh` — 启动脚本

每个文件的详细用法见 [中文模板指南](/resources/zh/templates/)。
