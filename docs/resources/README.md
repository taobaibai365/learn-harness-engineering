## 这里能找到什么

- 根指令模板
- 功能状态与进度文件
- 会话交接与收尾检查清单
- 启动脚本参考
- initializer 与 coding-agent 工作流说明
- 适合更长周期协作的 OpenAI 风格高级仓库骨架

## 推荐最小起步

- `AGENTS.md` 或 `CLAUDE.md`
- `feature_list.json`
- `claude-progress.md`
- `init.sh`

先从这四个文件开始，再根据项目复杂度补充 handoff、evaluator 和质量治理材料。
